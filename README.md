# Project Pumping Report
## Requirement
- php v-8.3
- check php -v
- node v-18
- check node -v
## Set up project
- composer install / composer update
- npm install
- create file .env lalu copy file .env.example ke dalam .env 
- php artisan key:generate
- php artisan migrate
- php artisan db:seed
- php artisan serve
- npm run dev
